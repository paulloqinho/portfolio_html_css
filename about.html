Sobre mim

Sou apaixonado por tecnologia e por este motivo busquei a transição de carreira para área que sempre amei, porém nunca tive a verdadeira coragem e hoje realizando a transição sei que fiz a melhor escolha. Atualmente estou cursando ensino Superior de Análise e desenvolvimento de sistemas utilizando como tecnologias C, C++, C#, PHP, Python, Java e Javascript, pela Universidade Veiga de Almeida - UVA- RJ, paralelamente estou me dedicando aos cursos e formações da plataforma de cursos ALURA voltados ao desenvolvimento profissional Front-end como Lógica de programação, HTML5, CSS3,Javascript,Typescript, React, GIT e GITHUB para adquirir e aperfeiçoar meus conhecimentos de programação para me tornar um desenvolvedor de qualidade.

No momento tenho conhecimentos de Front-End como: Lógica de Programação, HTML5, CSS3 e JavaScript. Meus próximos passos são: Estudar Git e Github, Typescript e React e continuar me aperfeiçoando em tecnologias de Front-End, para atingir o objetivo de aplicações de alta performance e qualidade.
